Welcome to Shoutbox's documentation!
==================================

Shoutbox is a powerful, easy-to-use Python client for the Shoutbox Email API. 
It provides a simple interface for sending emails with support for attachments, 
custom headers, and seamless integration with popular Python web frameworks.
